# Schemas

## Overview
This section contains schema-oriented documentation and structural definitions used across the kre8 demo repository.

## Purpose
Schemas provide a stable place to describe shared models and structural artifacts such as:
- Structured Intent
- Design Plan
- validation results
- future realm and law structures where appropriate
